Exerium Linux

Freedom, refined.

An Arch-based, Wayland-native Linux distribution built for those who demand speed, beauty, and complete control. Powered by Niri, Hyprland, and a custom Caelestia shell.

View on GitHub → Join Community

Built different.

Every detail considered. Nothing compromised.

⚡

Blazing fast

CachyOS kernel, zram, tuned I/O schedulers, and Wayland-native rendering. Boots fast, stays fast.

🔒

Private by default

Zero telemetry. Encrypted disk out of the box. Hardened browser. Tor-ready mode. Your machine, your data.

💎

Beautifully transparent

Custom glass theme, Caelestia shell, animated boot — and a system you can fully see and control.

🧩

The exr package manager

Atomic snapshots, instant rollback, AUR bridge, Flatpak support, and a declarative system config.

🛠️

Developer-ready

Toolchains, containers, terminals, and editors preconfigured. Ship code on day one.

🌐

Yours forever

GPL-3.0, community-governed, mirrored on Codeberg. No corporate strings. No dark patterns. Ever.

Building in the open.

Exerium is in early development. There is no ISO to download yet — but the foundation is being laid. Follow along, give feedback, contribute. Every step is public.
